# 视觉设计规范

> 从 SKILL.md 拆出。颜色/字号详表与禁忌清单。

## 基础参数

| 项目 | 规格 |
|------|------|
| 尺寸 | 16:9（10" × 5.625"） |
| Logo | `{SKILL_DIR}/assets/bring_logo.png`（代码中通过 `bring.LOGO_PATH` 自动解析） |
| 字体 | `bring.FONTS.primary` / `bring.FONTS.en` / `bring.FONTS.numeric` |

## 配色系统

> 完整颜色常量见 `docs/bring-templates.md` 底部 `bring.C` 对象。

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色（深邃商务蓝） | `003591` | 标题、重点、色块 |
| 辅助蓝 | `5385C5` | 竖条装饰、次级标题 |
| 浅蓝 | `80AACD` | 弱强调、辅助层级 |
| 极浅蓝 | `BBCEE5` | 卡片底、低强调背景 |
| 正文深灰 | `383535` | 正文（不用纯黑） |
| 辅助灰 | `7F7F7F` | 辅助说明 |
| 信息灰 | `A7A7A7` | 页码、注释、弱信息 |

`ACCENT`、`SUCCESS`、`DANGER`、`GOLD` 等旧键名仍保留，但已映射到蓝灰体系，避免橙绿红破坏当前高级商务蓝风格。

**五色步骤**：`003591` → `5385C5` → `80AACD` → `BBCEE5` → `A7A7A7`

## 字体层级

| 层级 | 字号 | 颜色 |
|------|------|------|
| 封面标题 | 36pt 粗 | `FFFFFF` |
| 章节标题 | 32pt 粗 | `FFFFFF` |
| 内容页标题 | 24pt 粗 | `003591` |
| 一级内容 | 20pt 粗 | `003591` |
| 正文 | 14-16pt | `383535` |
| 辅助/注释 | 9-13pt | `7F7F7F` |

## 禁止事项

- 相邻页不要用同一版式
- 正文左对齐（只有标题居中）
- 新模板使用 `bring.FONTS` 语义键，不直接写死字体字面量
- 严格使用薄云配色，不自创
- 内容页必须有蓝色竖条和 logo
- 不做纯文字页，必须搭配形状/卡片
- 正文用 `383535`，不用纯黑
- 标题下不加装饰线

## PPT 结构规范

```
封面页（1页） └─ addCoverSlide
章节N：分隔页（1页） └─ addSectionSlide
章节N：内容页（2-4页） └─ addContentSlide + B类版式
章节N：案例页（0-2页，可选） └─ addCaseDivider + addContentSlide+addCaseBox
...（重复章节）
金句页（0-1页，可选） └─ addFullQuote
封底页（1页） └─ addBackCoverSlide
```

| PPT类型 | 总页数 | 章节数 |
|---------|--------|--------|
| 简短演讲（15分钟） | 8-12页 | 2-3章 |
| 标准课程（45分钟） | 20-30页 | 3-5章 |
| 完整培训（半天） | 40-60页 | 5-8章 |

封面必须有。**封底默认为两页**（v3.7.12 起）：倒数第二页为 `closingQuote` 金句页，最后一页为 `backCover`（仅含 thanks / 二维码 / 联系方式）。`meta.disableClosingQuote=true` 可退回单页封底。每章以分隔页开头。章节内版式多样化。<10 页可不分章节。输出到用户指定目录。
